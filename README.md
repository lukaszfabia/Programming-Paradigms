# Programming-Paradigms

To run Ocaml, open terminal and write:

    ocaml
    #use "filename.ml";;
    #quit;;

To run Scala, open terminal and write:

    scala
    :load filename.scala
    :quit


#### List of topis: 
* recursion and tail recursion
* matching pattern
* high order functions
* for, List.Range, operators
